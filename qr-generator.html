How to Use the QR Code Maker

  1. Enter content: Paste a URL, text, email, phone number, Wi-Fi string, or any content you want to encode.
  2. Design: Pick a dots style, eye style, choose solid or gradient colors, and optionally upload a logo.
  3. Adjust size: Set pixel size and quiet zone (margin) for optimal scanning—especially for print.
  4. Label: Add a short caption (e.g., “Scan to visit”) for clarity and accessibility.
  5. Download: Export as PNG/JPG/WebP for web or SVG for sharp print quality.

Best Practices & Tips

  • Contrast: Keep high contrast between foreground and background for better scanability.
  • Logo + Error Correction: If using a logo, choose Q or H error correction.
  • Quiet Zone: Maintain a clear margin around the QR (at least 10–20 px for print).
  • Test Scans: Always test using multiple devices and lighting conditions.
  • Use SVG for Print: Vector format keeps edges sharp on any size.

FAQs

Is Toolify free?

Yes — it’s free to use with no watermarks.

Do you store my data?

No. All generation happens locally in your browser; nothing is uploaded.

Which format should I download?

Use SVG for print, PNG or WebP for web, and JPG for photos/heavy images.

Does adding a logo affect scanning?

It can. Increase error correction to Q or H and avoid covering the eyes or critical finder patterns.

Can I print large posters?

Yes — export as SVG for infinite scaling and crisp edges.

Troubleshooting

  • Not scanning? Increase contrast, size, and quiet zone; avoid overly dense logos.
  • Blurry print? Download SVG or a higher pixel size.
  • Logo not showing? Re-upload as PNG with transparency; adjust logo size/margin.
