Understanding the Different Modes

This intelligent mode automatically chooses the best processing method based on your input. If you upload an image, it uses Vision mode. If you only paste text, it uses Text mode. This is perfect for most users who want a seamless experience.

Troubleshooting Common Issues

‚ùå Common Problems and Solutions

Image Not Loading

  • Check file format (supports JPG, PNG, WebP, GIF)
  • Ensure file size is reasonable (under 10MB recommended)
  • Try refreshing the page and uploading again

Poor OCR Results

  • Improve image quality and lighting
  • Ensure text is clearly visible and not blurry
  • Try using Vision mode instead for better results
  • Check that text has good contrast with background

AI Not Understanding the Problem

  • Provide additional context in the text input
  • Try different modes (Vision vs OCR)
  • Ensure the image clearly shows the problem
  • Rephrase or clarify the question if possible

Slow Processing

  • Check your internet connection
  • Reduce image file size if possible
  • Try using Text mode for faster processing
  • Wait for the current process to complete before trying again
